# Basic Chatbot

A simple rule-based chatbot built with Python.

This project is a basic chatbot application that interacts with users by responding to predefined inputs such as greetings, basic questions and exit commands. It was developed as part of my Python internship to practice core programming concepts and understand how conversational logic works.

## Project Overview

The chatbot works by taking user input and checking it against predefined conditions. Based on the input, it returns an appropriate response.

The conversation continues until the user types **"bye"**, which ends the program.

This project focuses on implementing basic chatbot logic using simple Python fundamentals.

## Features

* Responds to basic greetings
* Handles simple predefined questions
* Continuous conversation using loops
* Exit command to end chat
* Easy to understand and beginner-friendly structure

## Built With

* Python 3

## How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/basic-chatbot.git
```

Move into the project folder:

```bash
cd basic-chatbot
```

Run the program:

```bash
python basic_chatbot.py
```

## Example Conversation

```text
You: hello
Bot: Hi!

You: how are you
Bot: I am fine, thanks!

You: what is your name
Bot: I am a simple chatbot.

You: bye
Bot: Goodbye!
```
